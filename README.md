# Traffic Prediction Machine Learning

A machine learning-based web application that predicts traffic congestion levels using real-time traffic parameters. The system uses ML models to classify traffic conditions as **Low**, **Moderate**, or **High** traffic.

---

## 🚀 Features

- Traffic congestion prediction
- Low / Moderate / High traffic classification
- Interactive Streamlit web interface
- Prediction logging using CSV
- Machine Learning model integration
- Real-time user input handling

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- Machine Learning
- Git & GitHub

## ▶️ Run the Project

### 1. Clone Repository

```bash
git clone https://github.com/reversetoe/traffic-prediction-machine_learning.git
```

### 2. Navigate to Project Folder

```bash
cd traffic-prediction-machine_learning
```

### 3. Install Requirements

```bash
pip install -r requirements.txt
```

### 4. Run Streamlit App

```bash
streamlit run app.py
```

## 📊 Prediction Categories

| Traffic Score | Prediction |
|---|---|
| < 3000 | Low Traffic |
| 3000 - 5000 | Moderate Traffic |
| > 5000 | High Traffic |
